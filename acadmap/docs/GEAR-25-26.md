# UCSB GEAR 2025-2026 integration

**Source PDF:** [25-26_GEAR.pdf](https://engineering.ucsb.edu/sites/default/files/docs/25-26_GEAR.pdf)

AcadMap stores a structured snapshot in `data/ucsb/coe-catalog.json`. GEAR-fidelity quarter plans live in `data/ucsb/coe-majors/{slug}.json`. Interactive graphs use `data/seeds/ucsb-{major}.json`.

## CoE pillar (Wave 2 — complete)

| Layer | Path | Status |
|-------|------|--------|
| Catalog summary | `data/ucsb/coe-catalog.json` | 5 majors |
| GEAR detail + quarter totals | `data/ucsb/coe-majors/*.json` | 5 majors |
| Interactive roadmaps | `data/seeds/ucsb-*.json` | 5 majors |
| UI | `/schools/ucsb/engineering/[major]` | `CoeQuarterTimeline` + `MajorSheetRequirements` |

npm run validate:coe-details

## Agent delegation

| Agent | Input | Output |
|-------|--------|--------|
| **GEAR Research** | PDF + catalog.ucsb.edu | Update `coe-catalog.json`, `docs/GEAR-25-26.md` |
| **Data / Seeds** | `preparation_for_major` + grids | `data/seeds/ucsb-*.json` with nodes/edges |
| **Product** | Catalog JSON | `/schools/ucsb`, `/schools/ucsb/[major]` |
| **Graph** | Seed JSON | Positions, prerequisite chains, career nodes |

### Prompt: GEAR Research agent

> Parse UCSB 25-26 GEAR PDF. Update `data/ucsb/coe-catalog.json` for all 5 majors: preparation, upper-division required, elective units, sample electives. Cite page numbers.

### Prompt: Data agent (per major)

> From GEAR and department 4-year grid, create `data/seeds/ucsb-{slug}.json` with course nodes, prerequisite edges, and 3-4 career nodes. Set `roadmap_available: true` in catalog.

## Website routes

- `/schools/ucsb/engineering` — CoE hub with GEAR detail badges
- `/schools/ucsb/engineering/{major}` — GEAR requirements + 4-year quarter plan with units
- `/roadmap/ucsb/{major}` — interactive prerequisite graph (all 5 CoE majors)
